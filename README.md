## About

We've been quietly building **Shed**, a code editor designed to stay out of the way so you can focus on writing, reading, and understanding code. We believe the best tools are dependable, thoughtful, and simple enough to disappear into the work.

## The Project

This repository contains the source code for **Shed**. Our focus is straightforward: build a fast, reliable editor with a clean foundation that can grow over time without unnecessary complexity.

## Developing

### Prerequisites

- Node.js `v24.18.0`
- npm

### Build

```bash
npm install --ignore-scripts
npx gulp vscode
```
